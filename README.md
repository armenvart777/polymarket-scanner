# polymarket-scanner

Сканер спортивных ставок на Polymarket. Сравнивает коэффициенты с реальными результатами ESPN и находит выгодные позиции.

## Возможности

- парсинг активных рынков Polymarket (спорт)
- кросс-проверка с ESPN API (NHL, NBA, MLB)
- расчёт уверенности на основе счёта и времени
- уведомления в Telegram о выгодных ставках
- автозапуск через cron

## Стек

Python 3.10+, urllib, json (без внешних зависимостей)

## Запуск

```
python scanner.py
```
